# 🚀 DSA Visualizer Pro

A professional, high-performance Data Structures and Algorithms visualizer built with **Django** and **Modern CSS**. This project aims to help students and developers understand complex algorithms through interactive, real-time visualizations.

![UI Preview](https://via.placeholder.com/800x400/0f172a/38bdf8?text=DSA+Visualizer+Pro+Interface)

## ✨ Key Features

- **📊 Algorithm Workbench**: Real-time visualization of Sorting and Searching algorithms.
- **🧠 Data Structures Lab**: Interactive exploration of Linked Lists, Stacks, Queues, and Binary Trees.
- **⚡ Big O Analysis**: Instant complexity feedback (Time & Space) for every algorithm.
- **🎨 Modern UI/UX**: Sleek dark-mode interface with glassmorphism, built for high engagement.
- **🛠️ Backend Driven**: Uses a Django REST API to calculate algorithm steps, ensuring accuracy and scalability.

## 🛠️ Tech Stack

- **Frontend**: Vanilla JS (ES6+), Modern CSS3, Bootstrap 5, Bootstrap Icons.
- **Backend**: Python, Django, Django REST Framework.
- **Visuals**: SVG for Trees, Dynamic DOM manipulation for Arrays and Lists.

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/dsa-visualizer.git
   cd dsa-visualizer
   ```

2. **Setup Virtual Environment**:
   ```bash
   python -m venv venv
   source venv/bin/scripts/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install Dependencies**:
   ```bash
   pip install django djangorestframework
   ```

4. **Run the Server**:
   ```bash
   python manage.py runserver
   ```

5. **Visit**: `http://127.0.0.1:8000`

## 📂 Project Structure

- `visualizer/views.py`: Core algorithm logic and API endpoints.
- `visualizer/static/visualizer/scripts.js`: Frontend orchestration and animation engine.
- `visualizer/templates/visualizer/index.html`: Modern, responsive UI shell.
